# My-Personal-Sketch-Gallery-App

This is my personal Sketch Gallery App where I store and organize all of my sketches. I built this application using React Native and Expo as a way to showcase my artwork while also applying and practicing the mobile development concepts I have learned.

The app allows me to categorize my sketches, add new ones from my device gallery, preview them, and manage them easily. Building this project was both a learning experience and a fun creative process, as it combines my interest in art and technology.
