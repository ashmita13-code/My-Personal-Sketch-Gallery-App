# My-Personal-Sketch-Gallery-App

This is a mobile app that I built using React Native and Expo to showcase my own sketches.

The app has different categories like Anime, Realism, and Favorites where I can organize and view my drawings.
When a user taps on any sketch, it opens in full screen just like a phone gallery, giving a clean and immersive viewing experience.

I added smooth animations when categories appear and visual feedback when buttons are pressed to make the app feel interactive and polished.

I created this project to practice React Native concepts like navigation, animations, and UI design — while also building something meaningful using my own artwork.
